# 🚀 Getting Started

Clone the repository 📦

```bash
git clone https://github.com/SalahHassan202/skillbridge-app.git
```

Go to project folder 📂

```bash
cd skillbridge-app
```

Install dependencies ⚙️

```bash
flutter pub get
```

Run the app ▶️

```bash
flutter run
```

Build APK 📦

```bash
flutter build apk --release
```
