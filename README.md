## 🚀 **Getting Started**

```bash
# 1️⃣ Clone the repository
git clone https://github.com/SalahHassan202/skillbridge-app.git

# 2️⃣ Navigate to project folder
cd skillbridge-app

# 3️⃣ Install dependencies
flutter pub get

# 4️⃣ Run the app
flutter run

# 5️⃣ Build APK
flutter build apk --release
```
